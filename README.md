# homebrew-tap

Personal Homebrew tap for [@nickhartjes](https://github.com/nickhartjes).

## Usage

```sh
brew tap nickhartjes/tap
brew install <formula>
```

Or install in one shot:

```sh
brew install nickhartjes/tap/<formula>
```

## Formulae

| Formula | Description | Source |
|---------|-------------|--------|
| `lazyjust` | Lazy TUI for [`just`](https://github.com/casey/just) — browse, search, and run recipes without memorizing commands | [nickhartjes/lazyjust](https://github.com/nickhartjes/lazyjust) |

## Updating

Formulae are published automatically by each project's release CI on tag push. Do not edit `Formula/*.rb` by hand.
